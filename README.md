# OC资料库

这是一个本地使用的 OC 整理网站，用来保存角色文字设定、标签、头像/封面、设定分支、关系网和图库图片。

## 使用方式

直接打开 `index.html`，或者在这个文件夹里启动本地预览服务后访问：

```text
http://127.0.0.1:5173/
```

资料会保存在当前浏览器的本地数据库里。换浏览器、清理浏览器数据、或换电脑时，记得先点右上角的导出按钮备份 JSON 文件；之后可以用导入按钮恢复。

## 保存说明

资料会自动保存到当前打开方式对应的浏览器本地存储里，并额外写入一份本地快照用于恢复。

请尽量固定一种打开方式使用：一直直接打开 `index.html`，或一直使用同一个本地地址，例如 `http://127.0.0.1:5173/`。如果你换成另一个端口、换浏览器、移动文件夹后从新路径打开，浏览器可能会把它当成另一个网站，原来的资料就不会出现在当前页面里。

重要资料建议定期点右上角导出按钮备份 JSON 文件。想分享给别人看时，点右上角的分享页按钮，会下载一个独立的 `oc-share-日期.html` 展示页；把这个文件发给别人，对方用电脑或手机直接打开就能浏览你的 OC 资料、图片、漫画、时间轴和关系网。

如果想发“只读分享链接”，需要先把导出的 `oc-share-日期.html` 放到一个朋友能访问的网址上。打开这个线上分享页后，页面顶部的“复制只读链接”按钮会复制当前网址；朋友打开后只能浏览，不能编辑你的资料。直接在本机打开 `file://` 或 `127.0.0.1` 预览时，这个链接只对你自己的电脑有效。

## 已包含功能

- 新建、删除、搜索 OC
- 为角色本体选择头像，上传后可在方形取景框里选择显示区域
- 同一个 OC 下建立多个设定分支
- 按 🌟 / 🌟🌟 / 🌟🌟🌟 筛选
- 每个设定分支单独保存世界观、角色概念、背景故事、详细故事、图片和关系网
- 每个设定分支拥有独立时间轴，可记录年份、时期、章节或关键事件
- 时间轴事件可关联当前分支已保存的插图或漫画，并支持双向点击跳转
- 上传封面与图库图片，封面边框会跟随原图比例变化，横图竖图都完整显示，且图片跟随当前设定分支
- 图库、漫画页和插图区支持横向滑动浏览缩略图
- 点击图库图片、漫画页或插图进入大图浏览，大图里可用左右按钮、键盘方向键或滑动切换图片
- 为图库图片添加文字介绍
- 添加漫画模块，保存多页漫画和剧情简介
- 漫画保存后可再次点击编辑，编辑时可看到已上传图片
- 漫画图片支持拖入添加，也支持拖动调整页序
- 在漫画后添加插图模块，支持拖入添加、图片介绍和大图浏览
- 每个设定分支拥有独立关系网
- 关系网支持直接添加已有角色、任意两点连线，并可设置连线颜色、箭头朝向、实线或虚线；双向箭头会分成两段并可填写反向关系文字
- 关系网节点可拖动位置并保存，非主角角色可添加方形裁剪后的角色外貌图
- 关系网画布支持放大、缩小、拖动平移、重置视图、重置布局和自动整理布局
- 点击关系网中的关联角色头像可以跳到对应角色，点击线条可以跳到对应连线条目并编辑
- 自动保存
- 顶部撤销按钮会保留最近几次保存版本，可恢复误删角色、误改文字或误删图片后的状态
- JSON 导入 / 导出备份
- 导出可分享的只读 HTML 页面，分享页会适配手机屏幕；分享页上线后可一键复制只读链接
