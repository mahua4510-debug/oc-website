# OC资料库

这是一个本地使用的 OC 整理网站，用来保存角色文字设定、标签、头像/封面、设定分支、关系网和图库图片。

## 使用方式

直接打开 `index.html`

默认打开时，网站会先尝试读取同目录下的 `data.json`。如果存在 `data.json`，页面会进入只读分享模式；如果没有 `data.json`，会进入本地编辑模式。需要强制进入编辑模式时，在地址后面加：

```text
?edit=1
```

资料会保存在当前浏览器的本地数据库里。换浏览器、清理浏览器数据、或换电脑时，记得先点右上角的导出按钮备份 JSON 文件；之后可以用导入按钮恢复。

## 保存说明

资料会自动保存到当前打开方式对应的浏览器本地存储里，并额外写入一份本地快照用于恢复。

请尽量固定一种打开方式使用：直接打开 `index.html`。

重要资料建议定期点右上角导出按钮备份 JSON 文件。想分享给别人看时，点右上角的分享页按钮，会下载一个独立的 `oc-share-日期.html` 展示页；把这个文件发给别人，对方用电脑或手机直接打开就能浏览你的 OC 资料、图片、漫画、时间轴和关系网。

如果想发“只读分享链接”，需要先把导出的 `oc-share-日期.html` 放到一个朋友能访问的网址上，再把页面地址发给对方；朋友打开后只能浏览，不能编辑你的资料。

## 注意事项

不要动oc-website文件夹内任何文件

每次更新功能或行为后，都需要把更新内容同步写入本文件。

## 已知问题

- 关系网在“展开画布”状态下，鼠标滚轮和触控板双指缩放在实际使用环境中仍可能完全不生效。目前已确认该问题尚未解决，暂时停止继续处理，后续重构关系网交互时需要重新检查事件监听、画布重绘和浏览器手势兼容性。
- 当前不要把工具栏显示的缩放百分比或代码中的滚轮事件绑定视为双指/滚轮缩放已经可用；后续修复需要使用真实鼠标和触控板完成验证。

## 最近更新

- 导出的只读分享展示页不再渲染“复制只读链接”按钮及其提示文案；只读模式、数据读取和其他展示功能保持不变。
- 曾尝试修复关系网双指/滚轮缩放，将监听改为绑定在稳定画布容器上并避免连续手势期间重建 SVG；实际环境中问题仍存在，详见“已知问题”。
- 关系网工具栏新增实时缩放百分比与“展开画布”按钮；点击画布本身不会展开，展开和退出改为明确按钮操作，并支持按 `Esc` 退出。
- 编辑页脚本和样式增加版本标记，避免浏览器继续使用旧缓存而导致关系网修复未生效。
- 关系网升级为白板/脑图式编辑体验：画布平移和节点位置保存已接入；鼠标滚轮、触控板双指缩放仍存在兼容问题。
- 关系网支持双击空白处弹出快捷菜单，可在点击位置新建角色节点或添加已有角色节点。
- 关系网节点新增连接点，可从连接点拖拽到另一个节点创建连线，创建后自动进入连线编辑。
- 关系线编辑增强：支持关系文字、方向、颜色、粗细、实线/虚线、直线/曲线/折线和箭头样式；箭头更醒目，关系文字有独立背景，避免被线条或节点遮挡。
- 关系网画布选择、右侧表单和列表现在会同步更新；导出的只读分享页会显示相同节点布局、视图缩放、线条样式、箭头和关系文字背景。
- 页面装饰新增模块外观自定义：可统一调整编辑模块和分享页内容卡片的背景颜色、背景透明度、边框颜色、边框透明度、边框粗细、圆角、阴影强度和玻璃模糊程度。
- 模块透明度只作用于模块背景，不会让文字、图片、按钮和边框一起变透明；模块边框独立保持清晰，避免背景透出后页面难以阅读。
- 页面装饰已重构为全局装饰系统：背景显示在整个网站/分享页最底层，正常内容叠在中间，贴纸显示在最上层。
- 背景支持上传、清除、铺满、完整显示、重复纹理、拉伸、透明度、模糊度和浅色遮罩，设置会实时作用到真实页面。
- 贴纸支持上传、拖拽移动、缩放、旋转、透明度、删除和上移/下移层级，位置按整页坐标保存到当前设定分支。
- 导出的只读分享页会显示当前分支保存的全局背景和贴纸装饰。

## 已包含功能

- 新建、删除、搜索 OC
- 为角色本体选择头像，上传后可在方形取景框里选择显示区域
- 同一个 OC 下建立多个设定分支
- 按 🌟 / 🌟🌟 / 🌟🌟🌟 筛选
- 每个设定分支单独保存世界观、角色概念、背景故事、详细故事、图片和关系网
- 每个设定分支拥有独立的全局页面装饰，可为完整页面和分享页设置背景与贴纸
- 每个设定分支可统一自定义模块外观，让编辑模块和分享页内容卡片支持半透明、清晰边框、圆角、阴影和玻璃模糊
- 每个设定分支拥有独立时间轴，可记录年份、时期、章节或关键事件
- 时间轴事件可关联当前分支已保存的插图或漫画，并支持双向点击跳转
- 上传封面与图库图片，封面边框会跟随原图比例变化，横图竖图都完整显示，且图片跟随当前设定分支
- 图库、漫画页和插图区支持横向滑动浏览缩略图
- 点击图库图片、漫画页或插图进入大图浏览，大图里可用左右按钮、键盘方向键或滑动切换图片
- 为图库图片添加文字介绍
- 添加漫画模块，保存多页漫画和剧情简介
- 漫画保存后可再次点击编辑，编辑时可看到已上传图片
- 漫画图片支持拖入添加，也支持拖动调整页序
- 在漫画后添加插图模块，支持拖入添加、图片介绍和大图浏览
- 每个设定分支拥有独立关系网
- 关系网支持白板式缩放和平移、双击空白处添加节点、从节点连接点拖拽牵线，并兼容旧关系网数据
- 关系网连线可设置文字、反向文字、颜色、粗细、箭头朝向、实线/虚线、直线/曲线/折线和箭头样式；关系文字带独立背景
- 关系网节点可拖动位置并保存，非主角角色可添加方形裁剪后的角色外貌图
- 关系网画布支持鼠标滚轮和触控板双指缩放、拖动平移、重置视图、重置布局和自动整理布局
- 点击关系网中的关联角色头像可以跳到对应角色，点击线条可以跳到对应连线条目并编辑
- 自动保存
- 顶部撤销按钮会保留最近几次保存版本，可恢复误删角色、误改文字或误删图片后的状态
- JSON 导入 / 导出备份
- 可读取仓库中的 `data.json` 和 `images/` 作为 GitHub Pages 默认只读内容
- 导出可分享的只读 HTML 页面，分享页会适配手机屏幕
